# Ассеты

Исходные изображения заказчика перенесены из корневой папки `images` в `apps/web/public/images/brand`.

## Файлы

- `header.jpg` — изображение для шапки публичной страницы.
- `background-example.jpg` — пример фона, пока не используется напрямую.
- `logo-ver1.jpg` — логотип, используется в публичной странице.
- `logo-ver2.jpg` — альтернативный логотип, используется на странице входа.
- `README.md` — исходный комментарий к изображениям.

Фронтенд обращается к ним по путям вида `/images/brand/header.jpg`.
