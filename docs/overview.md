# Обзор проекта

`snz-rodoved` — веб-приложение для переноса физических опросов в базу и анализа результатов.

Основной пользователь v1 — один рабочий/оператор, который вручную заносит анкеты и сам смотрит аналитику. Публичная страница существует по основному адресу, но не раскрывает статистику.

## Основные маршруты

- `/` — публичная страница с описанием проекта и контактным блоком.
- `/login` — вход в рабочую зону по паролю.
- `/editor` — ввод новых анкет.
- `/data` — фильтры, визуализация, таблица, редактирование и CSV-экспорт.
- `/admin` — отдельный вход администратора по логину и паролю.
- `/api/*` — API Fastify.

## Стек

- TypeScript
- React + Vite
- Fastify
- SQLite
- Drizzle ORM
- Zod
- Recharts

## Важные правила данных

- Анкеты не содержат ФИО, телефонов или других персональных данных респондента.
- Пропуск или нечитаемый ответ хранится как `unknown`.
- Вопросы 7 и 8 — отдельные поля, потому что оба есть в бумажной анкете.
- `q11WarDetails` заполняется текстом или `—`, если уточнения нет.
- Публичная страница в v1 содержит только минимальные шаблоны, пока финальное описание проекта не согласовано.
