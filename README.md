# Родовед: обработка анкет

Сайт для ручного ввода бумажных анкет и анализа интересов людей, которым нужна помощь с родословным поиском.

## Что входит в v1

- публичная страница проекта по адресу `/`;
- страница входа по паролю `/login`;
- админ-вход по логину и паролю `/admin`;
- страница ввода анкет `/editor`;
- страница работы с данными `/data`;
- ввод, редактирование, удаление и CSV-экспорт анкет;
- фильтры по дате, полу, возрасту, месту проживания и ответам;
- аналитика с графиками;
- SQLite-база с миграцией;
- документация в `docs/`;
- агентский handoff-слой в `AGENTS/`.

## Быстрый старт

```bash
npm install
cp .env.example .env
npm run dev
```

Локальные адреса:

- фронтенд: http://127.0.0.1:5173
- API: http://127.0.0.1:4000
- вход: http://127.0.0.1:5173/login
- админка: http://127.0.0.1:5173/admin
- ввод анкет: http://127.0.0.1:5173/editor
- данные и аналитика: http://127.0.0.1:5173/data

Если переменные не заданы, локальная разработка использует пароль `admin` для рабочей зоны и `admin` / `admin` для админки.
